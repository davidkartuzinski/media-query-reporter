Welcome to the Media Query Reporter Plugin. A simple plugin to help develope excellent responsive themes in WordPress.

== Description ==

The plugin uses a color coded bar, set to Media Query breakpoints to indicate the viewport width. It reports the height and width of the browser to help visualize exactly where the content changes.

Based and modified from a combination of:

https://github.com/davidcochran/CSS3-Media-Query-Reporter/blob/master/mediaquery-reporter.css 
 
and 
 
https://andylangton.co.uk/blog/development/get-viewportwindow-size-width-and-height-javascript.

== Usage ==

Install and activate the plugin on your development site, and then as you code the CSS or HTML, resize the browser to test and visualize the coding changes. Be sure to deactivate and delete this plugin on your production site. 

== Installation ==

Installing "Media Query Reporter Plugin" can be done either by searching for "Media Query Reporter Plugin" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the settings screen and configure, as desired.
